# ShefOS Reborn 🐧☢️

> Linux-дистрибутив на базе **Bomjaro**, созданный для мощного и удобного опыта.
> Идёт с широким набором предустановленных приложений — подойдёт и обычным пользователям, и профессионалам.



---

## ✨ О проекте

**ShefOS Reborn** — форк Bomjaro, заточенный под геймеров, разработчиков и просто
любителей пингвинов весом 320 кг. Вместо `systemd` используется собственный
`PENGVIN320KG_INIT`, который всегда пишет `Done !`, даже если что-то сломалось.

## 📦 Програмное обеспечение

- 🎮 Игры из коробки: Steam, PortProton, ElyPrismLauncher
- 🛠️ Полный тулчейн разработчика: gcc, g++, clang, python 3.14.3
- 🍷 Wine + Winetricks для запуска Windows-софта
- 🌐 Браузеров: Chromium, Brave, Opera / Opera GX, Yandex Browser, Vivaldi
- 💬 Мессенджеры и голосовая связь: Telegram, Mumble, TeamSpeak, MAX
- 🎨 Blender, GIMP — для творчества
- 📦 yay — AUR-хелпер из коробки
- ⚡ Вес системы: 320 GB (реально распаковывается в 15,8 ГБ)

## 📥 Установка

1. Скачать архивы из релизов
2. Распаковать
3. Запустить через QEMU

> ⚠️ Перед установкой убедитесь, что у вас есть 320 тонн свободног места на жёстком диске.
